Paddle Import

← back to Cairn

Pulls a route off PaddlePoints and drops it straight into your "Cairn GPX Routes" Drive folder — same folder Cairn already watches, so it shows up next time you hit "Check for new routes."

1. Capture a route

Drag this button to your bookmarks bar once. On any PaddlePoints route page, click it — this opens an import tab right away — then click PaddlePoints' own "View" button and the route gets sent over on its own.

If your browser blocks the tab from opening, allow popups for PaddlePoints and click the bookmarklet again.

⛺ Cairn: Grab Route

This only watches requests already made by the page you're on — it doesn't add any new network calls of its own, and it stops working the moment you leave or refresh that page.

2. Or paste it manually

Dev Tools (F12) → Network tab → click "View" on the route → click the admin-ajax.php request → Response tab → copy the full text and paste it below.
